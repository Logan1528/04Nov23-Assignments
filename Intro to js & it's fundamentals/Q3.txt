Q3: List and describe the five primitive data types in JavaScript.

primitive data types:

*Strings
*Numbers
*Booleans
*Undefined
*Null


Strings

    The string data type in JavaScript represents a sequence of characters that are 
surrounded by single or double quotes.

Number

    Number data type in javascript can be used to hold decimal values as well as 
values without decimals.

Booleans

    The boolean data type can accept only two values i.e. true and false.

Undefined
    
    The meaning of undefined is ‘value is not assigned’.

Null
    This data type can hold only one possible value that is null.
